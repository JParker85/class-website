Setup

At the beginning of our class we'll be taking some time to get a few things installed on you machines. If this is something you'd like to get a head start on here are a list of items you'll be needing.

Development Tools

Codepen

CodePen is a playground for the front end side of the web. It's all about inspiration, education, and sharing. Sign for a Codepen account.

OSX Mavericks

OS X Mavericks (version 10.9) is the tenth major release of OS X, Apple Inc.'s desktop and server operating system for Macintosh computers. OS X Mavericks was announced on June 10, 2013, at WWDC 2013, and was released on October 22, 2013, as a free update through the Mac App Store worldwide.

  1. You'll need to sign in with you Apple ID. If you don't have one, sign up here.
  2. Download the Mavericks upgrade from the Apple Store: download here.
  3. Double-click "Install OS X Mavericks" to begin installation.

WARNING: The Mavericks upgrade can take a bit of time to complete and will require a restart. Plan on doing this in the evening or over a lunch break.

XCode

Xcode is an integrated development environment (IDE) containing a suite of software development tools developed by Apple for developing software for OS X and iOS.

The important thing is that we install the XCode Command-Line Tools.

  1. Go to the Apple Developer Downloads site.
  2. Sign in with you Apple ID.
  3. Search for "Command Line Tools (OS X Mavericks) for Xcode - April 2014."
  4. Download the ".dmg" file for that package to your Desktop.
  5. Once downloaded, double-click on the file to unpack it.
  6. Double-click on the file called "Command Line Tools (OS X 10.9).pkg." and complete the guided installation instructions.

NOTE: If you have any issues with the download, not to worry! We'll have the files on hand and help walk you through the installation.

Homebrew

A package manager for installing developmental tools onto your Apple computer.

  1. Open up Terminal.app (Applications > Utilities > Terminal)
  2. Type the following command: 
    ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
  3. Follow the instructions on-screen.
  4. When finished, run brew doctor.

git

git is a distributed revision control and source code management (SCM) system with an emphasis on speed. Every Git working directory is a full-fledged repository with complete history and full version tracking capabilities, not dependent on network access or a central server.

git comes with your installation of XCode Developer tools. If you'd like to have more control over what version of git you'd like to use, run brew install git to install with Homebrew.

Github

GitHub is a web-based hosting service for software development projects that use the Git revision control system. Sign up for an account.

Creating an SSH key

You'll need an SSH key when using Github. SSH keys are a way to identify trusted computers, without involving passwords. Walk through the steps in the following tutorial to create your SSH key and add it to your Github account: https://help.github.com/articles/generating-ssh-keys

Atom

Atom is a free text editor you can use when writing, editing and crafting your code.

  1. Download Atom from the Atom website.
  2. After it's downloaded, move Atom to your Applications folder.
  3. Open Atom.
  4. Go to the Atom menu and click "Install Shell Commands".
  5. Go back to the Atom menu and click "Preferences".
  6. Click on "Packages" in the left-hand column.
  7. Search for "Lint" under Install Packages. Find the package called "Atom Lint".
  8. Click "Install" and wait for it to be installed.

NOTE: Sublime Text 3 is also an option, but will required a license purchase. Download here.

Google Chrome

Google Chrome is a web browser created by Google. We'll be using it for debugging, coding and general code reviews during class. Download here.

Node

Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient, perfect for data-intensive real-time applications that run across distributed devices.

Install node.js via Homebrew by typing brew install nodejs into Terminal.app.

Node Package Manager (npm)

npm is the official package manager for Node.js. It comes already installed when installing Node.js, but run through the following steps to make sure:

  1. Open Terminal.app (Applications > Utilities > Terminal)
  2. Type npm help
  3. You should see a list of usage commands listed out.

Sass

Sass is the most mature, stable, and powerful professional grade CSS extension language in the world. Use the following steps to install it to your local machine:

  1. Open Terminal.app (Applications > Utilities > Terminal)
  2. Type gem install sass. This will install Sass and any dependencies for you.
    (If you get an error message then it's likely you will need to use the sudo command to install the Sass gem: sudo gem install).
  3. You should now have Sass installed, but it never hurts to double-check. In your terminal application you can type: sass -v You should see a list of usage commands listed out.

Other Tools

Hipchat

Hipchat is a private group chat client that can be run on your desktop, browser or mobile phone. We'll be using Hipchat as our teams chat client throughout the class and a way for us to connect with eachother online.

Screenhero

Screenhero is an interactive screen sharing application. We'll be using it throughout the class for homework review and debugging during the lectures.

Alfred (Optional)

Alfred is a productivity application for Mac OS X, which aims to save you time in searching your local computer and the web. It can help you when doing useful things like: easily search for files and folders on your computer, quick web searches, access to your clipboard history and more!
